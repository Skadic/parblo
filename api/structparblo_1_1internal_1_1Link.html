struct Link#

Contains information about a link from a back-pointing block to its source block. This consists of the block’s index and the offset from which to copy.

Public Types

using IndexType = uint32_t#

The number type used by this class.

Public Functions

inline explicit constexpr Link(IndexType block_index)#

Creates a new source representing an invalid/non-existent source.

Parameters

block_index – The index of the block from which the link goes out, i.e. the index of the back block. This is the normal index on the current level.

inline auto is_valid() const -> bool#

Determines whether this source represents a valid block.

This merely checks whether the source_block_index is equal to the maximum of its numeric type.

Returns

true, if this refers to a valid block, false otherwise.

Public Members

const IndexType block_index#

The index of the block from which the pointer goes out on the current level of the tree.

IndexType source_block_index#

The index of the source block to which this link points on the current level of the tree.

IndexType offset#

The offset from which to copy from inside the source block.
