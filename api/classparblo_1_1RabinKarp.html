class RabinKarp#

A Rabin Karp rolling hasher.

Public Functions

inline RabinKarp(const uint8_t *source, const size_t string_len, const size_t window_size, uint64_t remainder)#

Create a new Rabin-Karp hasher from a pointer, with a given string length and window size with a pre-calculated remainder.

Parameters

  • source – A pointer into the string to be hashed.

  • string_len – The number of characters from the given pointer until the end of the string.

  • window_size – The number of characters each rolling hash will be hashing.

  • remainder – The remainder of BASE^window_size modulo PRIME

inline auto hashed_slice() const -> HashedSlice#

Returns the current state of the hasher in form of a HashedSlice object.

The contents of the object will be the pointer to the start of the currently hashed part of the string, the length being the window size and the hash value being the current hash value of the Rabin-Karp hasher.

Returns

A HashedSlice containing the current state of the hasher.
