Scan through the blocks pairwise in order to identify which blocks should be replaced with back blocks.

Parameters

  • bt – The block tree under construction.

  • s – The input string.

  • level – The current level.

  • is_adjacent – A bit vector detailing whether two blocks on the current level are adjacent or not. If a is_adjacent[i] is one, then blocks i and i+1 are adjacent.

  • block_starts – A vector holding the start position in the text for each block.

static inline void scan_windows_in_block_pair(RabinKarp &rk, RabinKarpMap<int> &map, MarkingAccessor markings, const size_t block_size)#

Scan through the windows starting in a block and mark them accordingly if they represent the earliest occurrence of some block hash.

The supplied RabinKarp hasher must be at the start of the block.

Parameters

  • rk – A Rabin-Karp hasher whose state is at the start of the block.

  • map – The map containing the hashes of block pairs.

  • markings – Contains the number of marks on each block.

  • block_size – The size of blocks on the current level.
