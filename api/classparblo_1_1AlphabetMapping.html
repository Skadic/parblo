class AlphabetMapping#

This class maps an extended ascii alphabet to a contiguous sequence of numbers starting at zero.

For example, given an alphabet a, c, d, f, g, we create a mapping for the alphabet \0, a, c, d, f, g which normally would correspond to extended ascii codes 0, 97, 99, 100, 102, 103. Note, that the null byte is always added to the alphabet even if it does not exist. This alphabet will be mapped to 0, 1, 2, 3, 4, 5.
